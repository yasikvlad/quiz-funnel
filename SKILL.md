# Quiz Funnel — генератор квиз-воронок

Скилл анализирует продукт, его аудиторию и создаёт полное техническое задание на персонализированный квиз-лендинг. Формула обратно спроектирована из лучших квиз-воронок (NeuroSamurai и аналоги).

## Что делает

1. Собирает информацию о продукте через диалог
2. Анализирует аудиторию (боли, триггеры, страхи, трансформации)
3. Генерирует полную структуру квиза: 20+ экранов, 8 вопросов, скоринг, калькулятор
4. Выдаёт готовое ТЗ в markdown — можно сразу дать Claude Code на реализацию

## Использование

```
/quiz-funnel Создай квиз для моего 21-дневного курса по продуктивности
/quiz-funnel Квиз-воронка для онлайн-школы английского, предзапись
/quiz-funnel Нужен квалификационный квиз для консалтинга по финансам
```

## Как работает

Скилл следует 5 фазам (подробности в PROMPT.md):
- **Фаза 1**: Сбор информации о продукте (интерактивно)
- **Фаза 2**: Анализ ДНК аудитории
- **Фаза 3**: Генерация структуры квиза по формуле 7 паттернов
- **Фаза 4**: Проектирование скоринга и калькулятора
- **Фаза 5**: Сборка полного ТЗ в markdown-файл

## Файлы

- `SKILL.md` — описание и триггеры (этот файл)
- `PROMPT.md` — полная инструкция по генерации квиза
- `references/formula.md` — формула 7 паттернов и карта экранов (справочник)
