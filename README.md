# Quiz Funnel v2.0 — генератор квиз-воронок для Claude Code

Скилл анализирует продукт и аудиторию, рекомендует оптимальный формат квиза из 4 вариантов и генерирует полное техническое задание на персонализированный квиз-лендинг.

Формула обратно спроектирована из 15+ высококонверсионных квиз-воронок (NeuroSamurai, Noom, Focus & Flow, Done., BedGear, RANAVAT и др.), дополнена исследованиями ADHD-UX и геймификации.

На выходе — markdown-файл, который можно дать Claude Code и получить рабочий Next.js квиз.

## 4 формата квизов

| Формат | Экранов | Время | Лучше всего для |
|--------|---------|-------|-----------------|
| **A. Диагностика** | 20+ | 3-5 мин | Курсы, спринты, челленджи |
| **B. Типология** | 12-15 | 2-3 мин | Блоги, инфо-продукты, digital |
| **C. Ультра-скрининг** | 5-7 | 1 мин | Консалтинг, SaaS, услуги |
| **D. Калькулятор-шоккер** | 8-12 | 1-2 мин | Детокс, продуктивность, финансы |

Скилл автоматически рекомендует формат на основе анализа продукта и ЦА.

| 1 | Чат-друг | Квиз имитирует диалог в мессенджере, не анкету |

| 3 | Цветовая шкала | Каждый ответ маркирован 🟢🟡🟠🔴 — визуальный удар |

| 6 | Калькулятор урона | Персональные цифры: сколько времени и денег теряешь |

| 8 | Зеркало проблемы | Вопросы как зеркало — пользователь осознаёт проблему |
| 9 | Cadence 3:1 | Каждые 3 вопроса — 1 мотивационная вставка (Noom) |
| 10 | Yes Ladder | Вопросы по нарастанию вовлечённости (Noom) |

## Что делает

Ты описываешь свой продукт. Скилл:

1. Задаёт уточняющие вопросы (ЦА, цена, формат, CTA)
2. Рекомендует оптимальный формат квиза из 4 вариантов
3. Генерирует ДНК аудитории (боли, триггеры, архетипы, трансформация)
4. Создаёт полную структуру квиза по формуле 10 паттернов с ADHD-friendly UX
5. Проектирует скоринг, калькулятор, архетипы (в зависимости от формата)
6. Собирает всё в одно ТЗ — готово к реализации

## Что на выходе

Markdown-файл с полным ТЗ:

- Выбранный формат квиза с обоснованием
- Дизайн-система (4 варианта: тёмная, calm wellness, energy business, nature)
- Стек (Next.js 14+ / Tailwind / Framer Motion / Vercel)
- JSON-структура всех экранов с полными текстами

- ADHD-friendly UX: без таймеров, чанкинг, прогрессивное раскрытие, мгновенная обратная связь
- Конверсионные механики: waitlist, milestone rewards, early bird, social proof
- CTA-библиотека от первого лица

## 5 фаз работы скилла

1. **Сбор информации** — диалог о продукте, ЦА, цене, формате, CTA
2. **Выбор формата** — автоматическая рекомендация A/B/C/D с обоснованием
3. **ДНК аудитории** — боли, триггеры, архетипы, трансформация
4. **Генерация квиза** — экраны, вопросы, вставки по формуле 10 паттернов
5. **Сборка ТЗ** — полный markdown-файл, готовый к реализации
