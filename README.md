# Quiz Funnel — генератор квиз-воронок для Claude Code

Скилл превращает описание любого продукта в полное техническое задание на персонализированный квиз-лендинг. Формула обратно спроектирована из лучших квиз-воронок рунета.

На выходе — markdown-файл, который можно дать Claude Code с командой "собери это" и получить рабочий Next.js квиз.

## Что делает

Ты описываешь свой продукт. Скилл:

1. Задаёт уточняющие вопросы (ЦА, цена, формат, CTA)
2. Генерирует ДНК аудитории (боли, триггеры, страхи, трансформация)
3. Создаёт 20+ экранов квиза по формуле 7 паттернов
4. Проектирует скоринг, калькулятор стоимости бездействия, категории результатов
5. Собирает всё в одно ТЗ — готово к реализации

## Формула 7 паттернов

| # | Паттерн | Что делает |
|---|---------|------------|
| 1 | Чат-друг | Квиз имитирует диалог в мессенджере, не анкету |
| 2 | Техно-фрейминг | Подан как "сканирование системы", не как тест |
| 3 | Цветовая шкала | Каждый ответ маркирован 🟢🟡🟠🔴 — визуальный удар |
| 4 | Микс вопросов | Single-select + multi-select + вставки с надеждой |
| 5 | Эмоциональная дуга | Любопытство → узнавание → шок → надежда → решение |
| 6 | Калькулятор урона | Персональные цифры: сколько времени и денег теряешь |
| 7 | Квалификация лида | Финальный вопрос сортирует горячих/тёплых/холодных |

## Структура квиза

```
ВХОД (2 экрана)
├── Квалификационный вопрос-фильтр
└── Эмпатия + валидация

ФРЕЙМИНГ (1 экран)
└── "Давай просканируем твою [систему]"

ДИАГНОСТИКА (8 вопросов + 1 вставка)
├── Q1: Ежедневное поведение
├── Q2: Количественный маркер ← основа калькулятора
├── Q3: Симптомы (multi-select)
├── ВСТАВКА: научный факт + надежда
├── Q4: Мотивация к действию
├── Q5: Потери за год (multi-select)
├── Q6: Провокационный сценарий
├── Q7: Текущее состояние
└── Q8: Готовность (квалификация лида)

РЕЗУЛЬТАТЫ (4 экрана)
├── Персональный дашборд (скор + 3 метрики)
├── Калькулятор стоимости бездействия
├── Влияние на качество жизни
└── Шоковое число (X ДНЕЙ за 10 лет)

ПРОДУКТ (3 экрана)
├── Обещание трансформации
└── Доказательства (исследования, статистика)

КОНВЕРСИЯ (2 экрана)
├── Финальный CTA + дефицит
└── Форма (2 поля)
```

## Установка

Скопируй папку в директорию скиллов Claude Code:

```bash
# Клонировать
git clone https://github.com/yasikvlad/quiz-funnel.git ~/.claude/skills/quiz-funnel

# Или скачать zip и распаковать в ~/.claude/skills/quiz-funnel/
```

## Использование

```
/quiz-funnel Создай квиз для моего 21-дневного марафона по продуктивности
/quiz-funnel Квиз-воронка для онлайн-школы английского, предзапись
/quiz-funnel Квалификационный квиз для консалтинга по финансам, чек 300к
```

Скилл задаст вопросы о продукте, сгенерирует ТЗ и сохранит файл `quiz-funnel-[название].md` в текущей директории.

## Что на выходе

Markdown-файл с полным ТЗ:

- Дизайн-система (тёмная тема, зелёный акцент, mobile-first)
- Стек (Next.js + Tailwind + Framer Motion)
- JSON-структура всех экранов с полными текстами
- Формулы скоринга и веса каждого ответа
- 4 категории результатов с порогами
- Калькулятор стоимости бездействия с формулами
- Список UI-компонентов
- Спецификация навигации и состояния
- Интеграция формы

Файл самодостаточный — Claude Code может собрать рабочий квиз без дополнительных пояснений.

## Скоринг

Детерминированный, без ИИ:

```
основной_скор = (сумма_баллов / 15) * 10    # от 0.0 до 10.0

Варианты: 🟢 = 3 балла, 🟡 = 2, 🟠 = 1, 🔴 = 0

Категории:
  8.0-10.0  🟢 Всё в порядке
  5.0-7.9   🟡 Предупреждение
  3.0-4.9   🟠 Серьёзная проблема
  0.0-2.9   🔴 Критично
```
